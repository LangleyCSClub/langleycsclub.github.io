Gitpod

Gitpod a cloud-based IDE that allows you to code in the browser. It is a great tool for beginners to get started with coding. It's a free tool that only requires a Github account to get started. It uses a linux container that allows you to run any language you want.

1. To get started, you have to create a Github account.
2. If you already have a Github account, you can skip this step.
3. Then create a Blank Github Repository.
4. Link your Github account to Gitpod.
5. Open your Github Repository in Gitpod.
6. Start coding!
